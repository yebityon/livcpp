# livcpp
**Li**bary for **v**vataarne written in **CPP**

## ライブラリ構成
* 必ず使う → `src/basic.cpp`
* 日付 → `src/date/`
* グラフ → `src/graph/`
* 数学 → `src/math/`
* 文字列 → `src/string/`

### その他
* いろいろ → `src/general/`
* C++14が使えない残念なジャッジシステムに使う → `src/cpp11/`
* VSCode用スニペット生成スクリプト → 'tojson.js'
* コード整形ルール → '.clang-format'
